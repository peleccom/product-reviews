# Installation

## Stable release

To install product-reviews, run this command in your terminal:

```sh
uv add product-reviews
```

Or if you prefer to use `pip`:

```sh
pip install product-reviews
```

## From source

The source files for product-reviews can be downloaded from the [Github repo](https://github.com/peleccom/product-reviews).

You can either clone the public repository:

```sh
git clone git://github.com/peleccom/product-reviews
```

Or download the [tarball](https://github.com/peleccom/product-reviews/tarball/master):

```sh
curl -OJL https://github.com/peleccom/product-reviews/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd product-reviews
uv pip install .
```
